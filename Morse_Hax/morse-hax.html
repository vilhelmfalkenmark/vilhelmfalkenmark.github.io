Vi går första året på Nackademins Yrkeshögskolas program Front-End Utveckling.

Vilhelm älskar att cykla och tycker att Van She är ett underskattat band.
Kodstyrkor: Loopar och validering
Kodsvagheter: Inga.

Fredrik har ett respektingivande skägg och tycker köttbullar med potatismos är den bästa maträtten.
Kodstyrkor: Manipulera Arrayer.
Kodsvagheter: Inga här heller.

Vi bestämde oss för att vi ville undersöka exakt hur svårt det var att översätta text till morse. Det tog cirka en timme att formulera problemet och strukturera upp den grundläggande logiken. Sen tog det cirka en halv-dag att åstadkomma översättningsfunktionen från vanlig text till morse. När vi var klara med det så bestämde vi oss för att det hade varit ännu mer utmanande att försöka översätta Morse tillbaka till vanlig text. Detta skulle visa sig något mer komplicerat. Mycket hår slets och en ansenlig mängd nikotin förtärdes. Det stora genomslaget kom när Fredrik knäckte följande javascriptrader:
var morseArray = []; var x = 0; var y = 1; for(i = 1; i <= breakPoint.length; i++) { if(i == 1){ morseArray.push(morseInputField.substring(0,breakPoint[x])); } else { morseArray.push(morseInputField.substring(((breakPoint[x])+ 1), ((breakPoint[y])))); x++; y++; }
Hade ovanstående kod varit en facebook-status hade den fått mer likes än en annonsering om nalkande barnafödande. I vilket fall. När vi var färdiga med ovanstående funktion så kom vi på att det hade varit ännu mer spännande att skapa ett eget "krypteringspråk". Som tur var kunde många av principerna återåteranvändas från Morseprojektet. Dock så krävdes några modikationer. Bland annat eftersom det som illustrerar mellanslag och ny bokstav varierar i stränglängd och teckensammansättning eftersom användaren själv anger en siffra för respektive strängsammansättning. Det stora genomslaget i det här projektet var när Vilhelm löste problemet med hur programmet ska kunna identifiera vad som är en ny bokstav med hjälp av följande javascript rader:
var breakPoint = []; for (var j = 0; j < haxToText.length; j++) { if (haxToText[j] == breakPointToString.charAt(0)) { for (var h = 0; h < breakPointToString.length; h++) { if (haxToText[j] == breakPointToString[h] && haxToText[j + 1] == breakPointToString[h + 1] && haxToText[j + 2] == breakPointToString[h + 2] ) { breakPoint.push(j); } } } }
När programmet till slut fungerade som det skulle var tårarna inte långt borta.

Det var svårt och genererade mycket frustration men det löste sig!

Hax översättaren

Hur fungerar det?
